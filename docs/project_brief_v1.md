# Project Brief v1

## Working title
ЗАХВАТ / «Спецназ: Точечный удар»

## Game concept
Top-down 2D tactical shooter with stealth elements where the player commands a 2-4 operator special-forces squad in modern combat scenarios.

## Core pillars
- tactics over speed
- squad coordination
- stealth or assault choice
- readable top-down battlefield
- realistic consequences

## Mission archetypes
- hostage rescue
- sabotage
- defense
- VIP escort

## Key systems implied by the brief
- direct control of one operator
- squad command system
- cover and line-of-fire logic
- visibility cone / stealth perception system
- noise and distraction system
- realistic damage consequences
- loadout and role selection
- mission rating
- persistent upgrades/meta progression
- procedural level assembly elements

## Roles
- assaulter
- sniper
- sapper
- support/generalist as needed

## Command examples
- follow me
- hold position
- cover fire
- move to point
- interact
- throw grenade to point

## High-risk implementation zones
- ballistic logic
- friendly fire
- hostage/VIP safety
- stealth detection
- save schema evolution
- top-down tactical readability on Android
