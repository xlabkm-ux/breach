## 1. Overview
- **Roles:**
  - **Patroller:** Moves on waypoints, sounds the alarm.
  - **Guard:** Static, holds narrow sectors.
  - **Anchor:** Protects the hostage, uses them as a live shield.
  - **Assaulter:** Flanks the player aggressively upon alert.
